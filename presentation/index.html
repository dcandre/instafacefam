Azure Cognitive Services? What's that?

  • Simple API calls that allow you to see, hear, speak, search, and understand content from your apps.
  • Can be deployed on Azure or run in a Docker container.
  • Secure and compliant set of services.

What APIs are Available?

  • Decision

    Content Moderator, Anomaly Detector, and Personalizer

  • Language

    Immersive Reader, Language Understanding, QnA Maker, Text Analytics, Translator Text

  • Speech

    Speech to Text, Text to Speech, Speech Translation, Speaker Recognition

  • Vision

    Computer Vision, Face Detection, Form and Ink Recognition, and Video Indexer

  • Bing Web Search

    Image, News, Video, Web, Autosuggest, Search with Images

What can the Vision APIs tell me about my pics?

The Computer Vision API can detect:

  • Faces
  • Objects
  • Brands
  • Landmarks
  • Celebrities

Other Vision APIs Can:

  • Extract key-value pairs and table data from pictures of form documents
  • Recognize digital ink (pictures of text)
  • Index voice, vocals, and visuals from videos

What can the Content Moderator API do?

  • Checks text, image, and video content for material that is potentially offensive, risky, or otherwise undesirable
  • Contains Review APIs and a Review tool that can assign human reviewers to content
  • Allows you to create custom rules for your content. Like a custom list of words that you don't want your users to use
